## 浏览器初识内核（内核包括 渲染引擎和 js 引擎）

| 浏览器  | 内核（渲染引擎）      | js 引擎        |
| ------- | --------------------- | -------------- |
| chrome  | webkit (fork)-> blink | v8             |
| Safari  | Webkit                | JavaScriptCore |
| FireFox | Gecko                 | SpiderMonkey   |

2008 年，谷歌公司发布了 chrome 浏览器，浏览器使用的内核被命名为 chromium。
